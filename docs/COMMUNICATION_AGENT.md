# 📡 Communications Agent Specification

## 📌 Role & Responsibility

The **Communications Agent** formulates public emergency announcements, formats tactical responder radio telemetry, and broadcasts incident status alerts across responder networks.

---

## 🛠️ Key Capabilities
* Generates clear, non-panicked public safety bulletins for SMS/Web distribution.
* Maintains real-time audit trails of dispatched operational orders.
