# 🚛 Logistics & Supply Agent Specification

## 📌 Role & Responsibility

The **Logistics & Supply Agent** plans emergency transport routes, schedules supply convoys, monitors fuel reserves, and calculates shortest safe paths around hazard buffer zones using OSRM.

---

## 🛠️ OSRM Pathing Mechanics
* Communicates directly with the Open Source Routing Machine (OSRM) API to derive real-time travel times and distance matrices for heavy rescue vehicles.
