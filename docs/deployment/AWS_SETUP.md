# ☁️ AWS Cloud Infrastructure Setup Guide

## 📌 AWS Bedrock Setup Steps

1. Log into AWS Management Console (`us-east-1` region).
2. Enable model access for **Anthropic Claude 3** under AWS Bedrock Model Access settings.
3. Create an IAM user with `AmazonBedrockFullAccess` policies.
4. Export `AWS_ACCESS_KEY_ID` and `AWS_SECRET_ACCESS_KEY` to `.env`.
